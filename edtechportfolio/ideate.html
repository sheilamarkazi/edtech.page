Arduino Project Ideation Unit

PROJECT DESCRIPTION

This eLearning instructional unit methodically guides students through various brainstorming strategies. Learning is supported so that students will be able to use Arduino and circuitry components to build creative projects that address real-world problems.

This unit scaffolds five brainstorming activities and includes scenario-based learning, interactive formative assessments, job aids, group activities, and more. Ultimately, learners ideate a creative personal project that they can build to address a real-world problem.


TOOLS USED

Articulate Rise, HTML/CSS/JS, Amazon Web Services Simple Storage Service (AWS S3), and Google Workspace.
